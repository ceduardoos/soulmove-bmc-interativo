# SoulMove | Business Model Canvas Interativo

Apresentacao navegavel do modelo de negocio do **SoulMove**, proposta de
mobilidade sustentavel gamificada integrada ao ecossistema Soul Up.

## Experiencia

- Nove blocos classicos do Business Model Canvas.
- Proposta de valor em destaque no centro da composicao.
- Explicacoes completas por hover, clique ou foco via teclado.
- Modo estatico para exportacao em `/?static=1`.

## Tecnologias

- React
- Vite
- Tailwind CSS
- lucide-react

## Executar localmente

```bash
npm install
npm run dev
```

## Publicar na Vercel

Importe este repositorio na Vercel. O framework **Vite** sera identificado
automaticamente, usando o comando de build `npm run build` e a pasta de saida
`dist`.

## Entregaveis

As versoes finais em PDF, PNG 4K e SVG editavel foram preparadas localmente para
inclusao no material academico.
